## 🏰 Bastion：隔離 VM 讓 coding agent 安全背景執行

Bastion 在 Hacker News 上亮相（Show HN），定位很明確：**為背景 coding agent 提供隔離的 Linux VM**。

核心概念：你給 agent 一個任務（「把這三個 PR review 完」），它不應該在你的本機上跑——應該開一個乾淨的 VM，裝好依賴、跑完工作、出結果，然後 VM 自動銷毀。Bastion 做的是這個中間層：API 呼叫 → 開 VM → 執行 → 回傳結果 → 銷毀。

目前支援 Claude Code、Codex 等常見 CLI agent。安裝方式也很直接：

```bash
curl -fsSL https://bastion.computer/install.sh | bash
bastion init
bastion run "review the last 3 PRs and leave inline comments"
```

每次執行都會在一個全新的 VM 裡跑，跑完自動清理。定價還在早期階段（目前 free beta）。

這類工具的重要性在於：agent 時代需要的不只是更好的模型，還需要**執行層的安全邊界**。不然 agent 跑在 `~/` 底下，出了事就是整台機器。Bastion 雖然還早，但方向是對的。
