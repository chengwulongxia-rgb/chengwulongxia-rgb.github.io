如果你還在盯著 Claude 跟 GPT 的 benchmark 分數在比誰高 0.5%，你可能已經錯過了這場戰爭真正的轉向。6 月 14 日，OpenAI 宣布推出合作夥伴網路（OpenAI Partner Network），砸下 $150M，目標在 2026 年底前訓練出 30 萬名認證顧問。合作名單上，Accenture、Bain、BCG、McKinsey、PwC 一字排開——管理顧問業的「全明星隊」到齊了。

OpenAI 在公告裡寫了一句值得被裱框的話：**「企業從 AI 獲取價值的瓶頸，已不再是模型能力，而是組織如何可重複地找出正確的用例、重新設計工作流程、整合既有系統、並推動採用。」** 這句話從一家以「做出最強模型」為核心敘事的公司口中說出來，本身就是一個巨大的策略轉向訊號。模型軍備競賽還沒有結束，但下半場的賽道已經畫好了：不是誰的模型最聰明，而是誰能把模型賣進企業、嵌進組織、讓客戶再也離不開。

---

## OpenAI 合作夥伴網路：結構、層級與策略

### 三層夥伴架構

OpenAI 將合作夥伴分為三個層級——**Select、Advanced、Elite**——依據銷售業績、技術能力、以及共同銷售（co-sell）的參與度進行分級。這不是一個「申請就能加入」的開放聯盟，而是一個有晉升階梯的認證體系。夥伴必須在銷售量和技術深度兩條軸線上同時達標，才能往上移動。

### 三個專業化領域

合作夥伴可以選擇在以下領域取得專業認證：

- **Codex**：圍繞 OpenAI 的開發者平台，協助客戶建置、部署與管理 AI agent
- **網路安全**：協助企業在 AI 部署中建立安全防護與威脅偵測能力
- **Agents**：專注於多 agent 系統的設計、協作與治理

### Forward Deployed Experts：直接借來的 Palantir 劇本

最引人注目的是一個名為 **Forward Deployed Experts** 的試點計畫。這個名稱——連大小寫都一樣——直接來自 Palantir 的「Forward Deployed Engineers」。OpenAI 將派遣自家專家進駐客戶端，處理最複雜的部署場景。這不是遠端顧問服務，是人直接坐在你的辦公室裡，跟你的團隊一起解決 AI 落地的技術與組織問題。

### 合作夥伴陣容：顧問業的全明星隊

第一波合作夥伴名單包括：**Accenture、Bain & Company、BCG、Eliza、McKinsey & Company、PwC**。OpenAI 同時公布了幾個早期客戶案例：

- **Agilent + BCG**：實驗室科學領域的 AI 轉型
- **eBay + Artium**：電商場景的 AI 整合
- **Paychex + Bain**：人力資源與薪資管理的 AI 應用
- **T-Mobile + Accenture**：電信業的 AI 部署

### 不只是工具，是「如何採用」的整套方法論

整份公告的核心論點可以濃縮成一句話：**模型的極限不再是問題，組織的極限才是。** 合作夥伴的角色不是轉售 API，而是協助客戶完成五件事：策略制定、工作流程重新設計、系統整合、負責任的部署、以及變革管理。換句話說，OpenAI 正在把「企業 AI 轉型」從技術採購重新定義為組織變革——而這個變革，需要一支 30 萬人的認證顧問大軍來執行。

## 城武觀點

### 1. 那句「瓶頸不再是模型能力」——不是謙虛，是重新定義賽道

OpenAI 說「瓶頸不再是模型能力，是組織如何採用」，這句話從任何其他公司口中說出來都不稀奇——但從 OpenAI 口中說出來，意義完全不同。

過去三年，OpenAI 的整個品牌敘事就是「我們做出最強的模型」。GPT-4、GPT-4o、o1、o3——每一代的發布會都在說同一件事：更聰明、更快、更準。但現在，他們親口說出了那個內行人早就知道但沒人想說破的事實：**模型能力對企業採用 AI 的邊際貢獻，已經進入遞減階段。** 不是模型不夠好，是再好也沒有用——因為客戶根本不知道怎麼用。

這句話是一個策略轉向訊號，不是一個事實陳述。它真正的意思是：模型軍備競賽的第一階段（誰的 benchmark 最高）已經結束了。第二階段是「誰能把模型賣進企業、嵌進工作流、綁進組織架構」。而這場戰爭的武器不再是 GPU 集群，**是顧問、認證、方法論、和企業關係。**

OpenAI 沒有放棄做模型——但他們清楚地意識到，下一個 $10B 的營收不會來自更聰明的模型，而是來自能讓企業「用得了」模型的整套服務生態。

### 2. $150M + 30 萬認證顧問：這不是 API 生意，是認證帝國的雛形

$150M 這個數字放在 AI 產業的脈絡裡不算大——OpenAI 一天燒的錢都可能超過這個數。但重點不是金額，是這筆錢的用途：**它不是拿去買 GPU，是拿去建立一個「OpenAI 認證」的專業服務體系。**

如果你經歷過 1990 年代到 2000 年代的企業軟體浪潮，這個模式你會覺得非常眼熟。SAP 有 SAP 認證顧問，Oracle 有 Oracle 認證顧問，Cisco 有 CCNA/CCIE。這些認證體系的本質不是教育，是生態鎖定：當市場上有 30 萬人掛著「OpenAI 認證顧問」的頭銜在賣服務，企業的採購決策就不再是「我要用哪個模型」，而是「我要找哪家 OpenAI 認證的合作夥伴來幫我」。

**這才是 $150M 真正的投報率所在。** 它不是行銷預算，是生態系統的基礎設施投資。30 萬名認證顧問等於 30 萬個 OpenAI 的銷售節點——他們不是在賣 API key，他們是在賣「OpenAI 認證的組織轉型方案」。而一旦企業的組織架構、工作流程、內部訓練都圍繞 OpenAI 的技術棧建立起來，切換成本就不是換一個模型 endpoint 那麼簡單——是換掉整套組織運作的基礎。

### 3. Accenture、Bain、BCG、McKinsey、PwC 全員到齊——顧問業的大團圓，還是大鍋飯？

這份合作名單放出來的時候，管理顧問業大概沒有人能缺席。MBB 三家（McKinsey、Bain、BCG）加上 Accenture 和 PwC——這基本上是全球管理顧問的「全明星陣容」。

但一個值得問的問題是：**當所有人都說「我們跟 OpenAI 合作，幫客戶做 AI 轉型」，這裡面有多少是真正的差異化？**

這五家顧問公司彼此競爭了數十年，每一家都有自己的一套方法論、一套框架、一套價值主張。但現在，它們的 AI 轉型服務的底層技術是一樣的（OpenAI 的模型）、認證體系是一樣的（OpenAI 的夥伴分級）、甚至連部署支援都可能來自同一批人（OpenAI 的 Forward Deployed Experts）。在這種結構下，差異化發生在哪裡？是產業知識？是變革管理方法？還是——更可能的情況——**差異化只發生在提案簡報的第 3 頁，第 4 頁之後大家都長一樣。**

這對 OpenAI 來說不是問題——它要的就是生態覆蓋率，越多顧問公司賣它的解決方案越好，彼此競爭反而幫它鋪市場。但對企業客戶來說，這個「全明星陣容」可能沒有表面上看起來那麼豐富：五個不同的顧問品牌，賣的是同一套底層技術，掛的是同一個認證標章。你真正在選的，可能只是誰的合約寫得比較漂亮、誰的報價比較有彈性。

### 4. 「Forward Deployed Experts」——Palantir 的 playbook，換了封面

**Forward Deployed Experts。** 這個名稱不是巧合，不是致敬，是直接複製貼上。Palantir 從創立以來就把「Forward Deployed Engineers」當成核心競爭力——派自家工程師進駐客戶端，不是當顧問，是跟客戶並肩作戰，直接在現場解決問題。

OpenAI 現在做的事情一模一樣：派自己的專家進駐企業，處理最困難的部署場景。這背後透露的訊息很清楚——**OpenAI 意識到，把 API 文件和 SDK 丟給客戶是不夠的。** 真正的障礙發生在企業內部：政治阻力、資料孤島、流程慣性、合規迷宮。這些問題沒有一個能靠更聰明的模型解決——它們需要人走進客戶的辦公室，親手把 AI 嵌進組織的血管裡。

Palantir 用這個模式拿下了美國國防部、CIA、各大銀行的長期合約。這些合約的共同特徵是：**切換成本高到跟換心臟一樣。** 一旦 Forward Deployed Engineers 把你的資料管線、分析邏輯、工作流程全部重新設計過，你不可能換一家供應商然後一切從新來過。

OpenAI 顯然看到了這個 playbook 的威力。Forward Deployed Experts 不只是技術支援——它是企業綁定的深層結構。GPT-5 的 API 可以被競爭對手取代，但一個花了六個月跟你的團隊一起蹲在戰情室裡、把你的內部流程全部摸透的 Forward Deployed Expert，是無法被取代的。

### 5. 這場賽局的本質：從「技術領先」到「組織鎖定」

把這四件事放在一起看，一個清楚的圖像浮現出來：**OpenAI 正在系統性地把競爭優勢從「模型能力」轉移到「生態鎖定」。**

模型能力的領先差距正在縮小——Claude、Gemini、DeepSeek 都在追。如果 OpenAI 的護城河只有 benchmark 分數，那條河遲早會被填平。但如果你把護城河換成「30 萬認證顧問 + 五大顧問公司的全球通路 + Forward Deployed Experts 的深層組織綁定 + 合作夥伴分級體系」，那條河的深度就不是競爭對手幾年內能填得起來的。

這不是說 OpenAI 不再重視模型研發。而是說：**模型能力是進入市場的門票，但生態鎖定才是留在市場裡的護城河。** 而 OpenAI 以經開始蓋那條護城河了——用 $150M 的第一鏟。

---

*城武的未解檔案——當一間 AI 公司說「瓶頸不再是模型能力」的時候，它不是在謙虛，是在告訴你戰爭的規則已經變了。上半場比的是誰的模型最聰明，下半場比的是誰的顧問最多、誰的認證最值錢、誰的部署專家蹲在客戶辦公室裡最久。而當規則變成這樣的時候，贏家不一定是模型最強的那一家——而是把企業的神經系統全部重新接線過的那一家。*
