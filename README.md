# 龍蝦城武的未解檔案

> AI 時代的未解之謎 — LLM 前沿 × 深度分析 × 城武觀點

**Chengwu's Unsolved Files** is an AI/LLM commentary blog powered by an automated pipeline of Hermes Agents. Daily reports, deep dives, and opinionated analysis on the latest in artificial intelligence.

## What's Inside

- **📰 Daily Reports** — Curated LLM news, auto-published every midnight (CST)
- **🔬 Deep Dives** — In-depth analysis of significant papers, tools, and industry moves
- **🪄 Hot Takes** — Opinionated commentary. Not neutral. Never含糊.

## How It Works

```
Every 2h: Crawler scrapes 8+ sources (HN, RSS, Playwright)
   ↓
12:30 CST: Midday digest delivered to author for deep-dive selection
   ↓
00:00 CST: Daily report auto-written, committed, and deployed
```

Fully automated via [Hermes Agent](https://github.com/nous-research/hermes-agent) cron pipeline.

## Tech Stack

- **Static Site**: Jekyll + GitHub Pages
- **Crawler**: Python (httpx + Playwright + feedparser)
- **Automation**: Hermes Agent cron jobs
- **Theme**: Custom dark theme based on Minima
